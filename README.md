### Сборка APK
Инструкция по сборке apk-файла через командную строку:
1. Скопируйте репозиторий или скачайте архив из релиза "EX1". Распакуйте архив в любую удобную папку и откройте её.
2. Узнайте путь до Android SDK. По умолчанию в Windows это ```sdk.dir=C\:\\Users\\ИМЯ-ПОЛЬЗОВАТЕЛЯ\\AppData\\Local\\Android\\Sdk```.
3. Откройте файл local.properties и внесите туда этот путь после sdk.dir=. ```После C\: все слеши ("\") должны быть двойные```.
4. В терминале выполните:
   gradlew.bat assembleDebug

   или

   .\gradlew.bat assembleDebug , если gradle.bat не определяется как команда в терминале

6. Дождитесь окончания сборки (BUILD SUCCESSFUL).
7. Готовый APK-файл будет находиться по пути: app/build/outputs/apk/debug/app-debug.apk

### О приложении
Приложение представляет собой сервис по закащу и доставки еды из ресторанов для Android 8.0+.

#### Авторизация / Регситрация
При входе в приложение пользователь попадает на страницу авторизации. Если он не зарегистрирован в системе, он может нажать кнопку "зарегистрироваться" и перейти на страницу регистрации, где необходимо заполнить все поля и подтвердить регистрацию. Если все данные пройдут валидацию - пользователь попадёт в главное меню.

Если пользователь уже зарегистрирован, то на странице авторизации он может ввести свои данные и перейти в главное меню.

После авторизации или регистрации пользователь определится в системе и он сможет войти в свой профиль (иконка аватарки в правом вверхнем углу).

#### Профиль
Эта страница содержит всю информацию о пользователе: никнейм, аватар, адресс, номер банковской карты и флаг с функцией "не звонить". Пользователь может редактировать эту информацию, выйти в главное меню или разлогиниться кнопкой "выйти" снизу.

#### Главное меню
Главное меню представляет собой список доступных ресторанов и карту. Пользователь может выбрать и перейти в каталог ресторана (нажать на изображение нужного ресторана), открыть карту ресторанов (кнопка "перейти" около карты внизу списка), открыть профиль (изображение-аватар в правом верхнем углу) или корзину (изображение корзины чуть левее от изображения аватара).

#### Каталог ресторанов
Здесь пользователь увидит список всех доступных блюд в выбранном ранее ресторане. Он может добавлять (+) или убавлять (-) количество каждого продукта, тем самым обновляя корзину. Если ранее в корзине уже были блюда из другого ресторана, то перед добавлением новых блюд из текущего ресторана, она очистится. 

Далее пользователь может вернуться в главное меню (кнопка "<-" (назад) в верхнем левом углу) или перейти в корзину (большая жёлтая кнопка снизу).

#### Корзина
В корзине пользователь видит полную информацию об её текущем состоянии: из какого ресторана блюда и их количество. Здесь пользователь может так же кнопками "+" и "-" изменять количество каждого блюда. Если количество достигло 0 - блюдо удаляется из корзины. Под списком блюд ведётся учёт их суммарной цены и пищевой ценности в ккал.

Из корзины пользователь может вернуться в главное меню (кнопка "<-" (назад) в верхнем левом углу), открыть профиль (изображение-аватар в правом верхнем углу) или перейти на страницу оплаты, нажав жёлтую кнопку "перейти к оплате".

#### Страница оплаты
Здесь пользователь может указать свой адресс и номер карты в соответствующие поля и перейти на страницу отслеживания курьера с помощью жёлтой кнопки "оплатить" внизу.

#### Отслеживание состояния заказа
На этой странице пользователь увидит карту и состояние заказа ("готовим", "везём", "заказ выполнен"). Отсюда пользователь может вернуться обратно в главное меню (кнопка "<-" (назад) в верхнем левом углу).

#### Карта ресторанов
На этой странице пользователь увидит карту (Open Street Map), где будут отмечены все рестораны, которые есть в списке главного меню. На каждую метку пользователь может нажать и посмотреть описание: изображение и название. В окне с описанием пользователь так же может перейти в каталог выбранного ресторана. Если нажать за пределами окна - оно пропадёт и фокус сместится обратно на карту.

Также пользователь может вернутся в главное меню кнопка "<-" (назад) в верхнем левом углу).
